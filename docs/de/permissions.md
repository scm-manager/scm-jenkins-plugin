---
title: Berechtigungen
---
Das Jenkins-Plugin bringt folgende Berechtigungen mit: 

* Globale Berechtigungen:
    * Globale Jenkins Konfiguration bearbeiten: Darf die globale Jenkins Konfiguration ändern.
* Repository-spezifische Berechtigungen
    * Repository Jenkins Konfiguration: Darf die Jenkins Konfiguration des jeweiligen Repository ändern.
